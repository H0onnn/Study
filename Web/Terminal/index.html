--- #Terminal 명령어 --- VS code 터미널 단축기 cmd + j 1. ls: List를 의미하며 입력할 때 마다 현재 있는 디렉토리의 콘텐츠를 나열한다. 2. pwd: Print Working Directory = 현재 디렉토리 위치를 나타낸다. 3. cd: Change Directory = 디렉토리를 변경할 때 사용한다. - cd ~ = 홈 디렉토리로 이동 - cd directoryName = 해당 디렉토리로 이동 - cd .. = 직전 디렉토리로 뒤로가기 - cd /Users/jh/Desktop/Web = 해당 디렉토리로 이동 (절대경로), 모든 폴더 내에서 사용할 수 있음 - cd ../../Downloads = 해당 디렉토리로 이동 (상대경로), 현재 위치에 따라 작동 4. mkdir: Make Directory = 새로운 디렉토리를 생성한다. - mkdir folderName folder2 = (폴더명을 사이로 공백을 두어 한 번에 여러개를 만들 수 있다.) 5. touch: 새로운 파일을 생성한다. - touch index.html app.css app.js = mkdir과 마찬가지로 파일명 사이 공백을 두어 한 번에 여러개의 파일을 만들 수 있으며, 확장자를 입력 해야한다. 6. rm: 파일이나 폴더를 삭제한다. (휴지통으로의 삭제가 아닌 영구 삭제이므로 사용시 유의하자) - rm 파일명 = 파일 삭제 - rmdir 폴더명 = 폴더 삭제 (비어있는 폴더만을 삭제한다.) - rm -rf 폴더명 = 폴더 및 폴더 내 모든 내용을 삭제한다.
